
 [Ι] ΘΕΣΕΙΣ ΔΕΔΟΜΕΝΩΝ
 =====================

 Τα δεδομένα για τα Interface είναι χωρισμένα σε δύο σημεία:
 
   (1) Σε αυτά στα οποία απαιτείται συννεχόμενη πρόσβαση και τα οποία είναι πιό αποδοτικό το να βρίσκονται
       στο manifest της function.
	   
	   Τα δεδομένα που βρίσκονται εκεί αφορούν την επιλογή του καταλληλότερου τρόπου εμφάνισης των δεδομένων
	   ή του προεπιλεγμένο τρόπου εμφάνισης δεδομένων έτσι ωστε να μήν απαιτείται η προσθήκη του χρησιμοποιούμενου
	   interface στο URI της αίτησης.
	   
	   Αυτά τα δεδομένα είναι:
	   
	   	* default_interface 	Action's Default interface
		* forced_interface  	Use ONLY a specific interface (non-overriden by URL)
		* default_outmode		Action's Default output processor
		* forced_outmode		Use ONLY a specific output processor (non-overriden)
		* post-processor		Use an (overridable) second-level output processor
		
		
   (2) Και σε αυτά που βρίσκονται στον πίνακα interface_list, και τα οποία περιέχουν πιό λειτουργικές πληροφορίες, αλλα
       μικρότερης απαίτησης.
	   
	   Τα δεδομένα αυτά χρησιμοποιούνται κυρίως σε συνδιασμό με τα υπόλοιπα δεδομένα και βρίσκονται στην SQL για 
	   διασυνδεδεμένες αιτήσεις.
	   
	   Αυτά τα δεδομένα είναι:
	   
	   * guid					Interface Reference Unique ID
	   * name					Interface display name
	   * filename				Filename to use for displaying this interface
	   * positioning			Interface's possible positions:
	   
	   		-- POPUP			Can be popped up in a dedicated window
			-- MAIN				Can be displaied on the main window area
			-- FULL				Can be used to render a whole pacge
	
 [ΙΙ] ΜΕΝΟΥ, ΔΥΝΑΤΟΤΗΤΑ ΕΜΦΑΝΙΣΗΣ  
 ================================

  Το σύστημα βασίζεται πάνω στην ικανότητα να δημιουργούνται nested interfaces. Αυτό σημαίνει οτι πάνω σε ένα interface,
  ή μέσω ενός interface εμφανίζεται κάποιο άλλο interface. Οι πληροφορίες για αυτό το nesting βρίσκονται στον πίνακα
  interface_links. Του οποίου τα πεδία σημαίνουν:
  
  		* index					Α/Α
		* interface				Το GUID του interface
		* action				Το GUID της ενέργειας που μπορεί να εκτελεστεί μέσα απο αυτό το interface
		* position				Θέση στην οποία θα εμφανιστεί το interface όταν εκτελεστεί
								Η θέση αυτή εξαρτάται απο το host interface. Δές τον πίνακα 1 για πιθανές θέσεις
		* require_script		Όνομα του αρχείου Script το οποίο πρέπει να εκτελεστεί και να επιστρέψει true προκειμένου να 
								θεωρηθεί οτι η συγεκριμένη ενέργεια μπορεί να εμφανιστεί στο παρόν interface

  [Α] ΠΙΝΑΚΕΣ
 =============
 
   Πιθανές τιμές του πεδίου interface_links.position ανάλογα με το interface
   
   +---------------------+--------------------+------------------------------------------------------------------------------+
   |      INTERFACE      |      POSITION      |                                   MEANING                                    |
   +---------------------+--------------------+------------------------------------------------------------------------------+
   | Main Map Interface  |        POPUP       | Το interface θα εμφανιστεί σε dedicated popup παράθυρο μέσω του συστήματος   |
   |					 |                    | JSON. Οι παράμετροι που θα σταλούν, θα περιέχουν το m=jason				     |
   |					 |         MAIN       | Το interface θα εμφανιστεί στην κεντρική περιοχή πάνω στην οποία εμφανίζεται |
   |					 |  				  | ο χάρτης, καθαρίζοντας το interface cache									 |
   |					 |         INFO       | Το interface θα εμφανιστεί στην κεντρική περιοχή πάνω στην οποία εμφανίζεται |
   |					 |  				  | ο χάρτης, με την εξαίρεση οτι το προηγούμενο περιεχόμενο μεταφέρεται στο     |
   |			         |                    | interface cache, και μπορεί ο χρήστης να επιστρέψει στην προηγούμενη θέση    |
   |					 |         FULL       | Το interface θα εμφανιστεί σε ολόλκηρη τη σελίδα. Αυτό προϋποθέτει οτι το    |
   |					 |                    | script θα δημιουργήσει μόνο του έναν πλήρη HTML κώδικα						 |
   |					 |        BLANK       | Το interface θα εμφανιστεί σε ένα νέο κενό παράθυρο. Αυτό προϋποθέτει οτι το |
   |					 |                    | script θα δημιουργήσει μόνο του έναν πλήρη HTML κώδικα						 |
   | 					 |         ΝΟΝΕ		  | Όλες οι επιστροφές δεδομένων απο το interface οδηγούνται για διαχείρηση απο  |
   | 					 | 					  | κώδικα. Κανένα αποτέλεσμα δέν εμφανίζεται									 |
   |					 |         GRID	      | Το μήνυμα είναι αντικείμενο JSON δομής κανάβου και χρησιμοποιείται απο java  |
   |					 |					  | για την εμφάνιση του κανάβου του παιχνιδιού									 |
   |                     |       DEDICATED    | To Interface θα εμφανιστεί σε συγκεκριμένο παράθυρο το οποίο θα εμφανιστεί   |
   |					 | 					  | κεντρικά, αποκλειστικά
