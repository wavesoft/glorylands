GloryLands Map Editor Extended Tutorial

Introduction

MapEditor also offers some usefull actions to simplify your design. For example, instead of clicking on the map you can use some keystrokes and mouse moves instead.

This tutorial will show you the remaining functions of the editor.

Using keystrokes while designing

Two special keys can be used while designing. The CONTROL and the SHIFT key. Their function varies, depending of the operation being executed:

  • While Designing, holding down SHIFT will enable the overwrite mode. While in overwrite mode, you can place an object anywere on the map (If you are not in overwrite mode clicking on an existing object will start the drag/resize mode)
  • While Designing, holding down CTRL will enable the hover mode. While in hover mode, each object is being highligthed when the mouse moves over it. When an object is highlighted, a horizontal line and a rectangle are displaied. The line represents the item's Z=0 position (Ground position or Z-Reference position) and the rectangle marks the area the current object occupies.
Fig.20 : An object in hover mode
  • While Setting Attennuation, Holding the SHIFT button will emulate the mouse click. So, if you have to fill a large area, you han hold down SHIFT and move your mouse over the area you want to fill.

Also, note that the ESC key will cancel any active operation (For example object repositioning or z-reference modification)

Modifying Z-Reference

There are some cases you will need to modify the z-reference line of an object. When placing an object, it's default z-reference is 0. That is tranlated to "the object is 0 units high from the ground" or simply, its on the ground. If you want to place an object on a higher altitute you will have to modify it's Z-Reference. For example, windows are not on the ground level, they are on a specific position above the ground. The next image shows a wrong window z-reference:

 
Fig.21 : Windows having an invalid Z-Reference position (They are on ground-level) Fig.22 : The compiled result of the previous example. You can see that the windows are placed inside the building instead in front of it.

To change some object's Z-Reference use the object's dropdown menu and select "Change Z-Reference". After the click, the object is highlighted and the horizontal Z-Reference line starts floating. Left-click on the new position to save the item's Z-Reference or hit ESC to cancel.
