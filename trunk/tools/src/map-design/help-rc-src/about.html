About GloryLands Map Editor

About the game

GloryLands is a web-based massive multiplayer online RPG game that is based on a community contribution system. Every one can submit it's own maps, quests, items, stories or suggest for new actions, skills, proffessions etc., and thus the game could be considered "endless".

About the author

The whole system is designer and coded by John Haralambidis (johnys2@gmail.com).

The game graphics are mostly obdained from various on-line resources but some of them are designed from scratch.
