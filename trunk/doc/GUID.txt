
	 Το GUID είναι ένας αριθμός. Του οποίου το low-order byte είναι αυτό το οποίο δηλώνει τον
	 τύπο τους, και τα υπόλοιπα byte το index του. Επιπλέον, το πρώτο bit του πρώτου byte δηλώνει
	 άν το GUID αναφέρεται σε Instance ή σε Template. Έτσι:
	 
		 	 	        xxxxxxxxxxxxxxxxxxxxxxxx    nnnnnnn       T
							24 bit = 16777216     7bit = 128 	1 = Instance
								(Item Index)	  (Category)	0 = Template
    
	Η κατηγορία που δηλώνει το Low-order byte του GUID αντιστοιχεί σε όνομα πίνακα μέσω του λεξικού GUID, τα 
	δεδομένα του οποίου βρίσκονται σε cache μορφή στο αρχείο engine/data/guid_dictionary.php και σε SQL μορφή
	στον πίνακα system_group_dictionary.
	
	Ωστόσω, μιάς και το range αυτό είναι αρκετά μεγάλο, απο την λίστα των 128 category εξαιρείται το 
	τελευταίο range τών 64 κατηγοριών, οι οποία αντιστοιχεί σε ειδικά εσωτερικά GUID. Πιό συγεκριμένα:
	
	Κατηγορίες:
	
	* 0~64		= Χρήση για τους πίνακες ?_instance, ?_template, ?_vardesc
	* 65~127	= Χρήση απο το σύστημα (Ακόμα δέν υπάρχει κάτι)
